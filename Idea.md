# Проект: option-pro (ETH Options Dashboard)
## Контекст и Техстек
Backend: Golang. Фокус на производительности и обработке данных в реальном времени.

Frontend: Next.js. Современный, быстрый интерфейс.

Visualization: D3.js. Нам нужны сложные тепловые карты (heatmaps) и графики волатильности для опционов ETH.

Инструментарий: OpenClaw (шлюз) и Ollama (локальные модели).

## Доменная область (ETH Options)
Цель: Анализ опционов на Ethereum (ETH).

Ключевые метрики: Греки (Delta, Gamma, Theta, Vega, Rho), подразумеваемая волатильность (IV), объем и открытый интерес.

Визуализация: Построение поверхности волатильности и интерактивных досок опционов.

## Инструкции по коду
Golang: Пиши чистый, идиоматичный код. Используй типизацию для структур данных опционов.

D3.js: Мы делаем упор на производительность рендеринга больших объемов данных.

Архитектура: Разделяй логику обработки данных (Backend) и визуальную составляющую (Frontend).

## Приоритеты задач
Разработка архитектуры дашборда.

Создание тепловых карт (heatmaps) для визуального анализа рынка.

Интеграция данных по ETH опционам в реальном времени.
